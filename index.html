1. Introduction

Coretex ("the Bot") is developed and maintained by Coretex Studios. This Privacy Policy explains what data we collect, how it is used, how it is protected, and what rights users have regarding their data. By using Coretex, you agree to the terms described here.

2. Data We Collect

Automatically Collected

  • Discord User IDs
  • Server IDs
  • Channel IDs
  • Message Content (only for moderation, logging, tickets, or features that require it)
  • Interaction data (commands, button clicks, event usage)

User-Provided Data

  • Server configuration (prefix, mod role, logging channels)
  • User & guild preferences
  • Ticket information
  • Quotes and stored messages

3. Why We Collect This Data

  • To operate bot features effectively
  • To provide moderation, ticketing, logging, and automation
  • To save server settings and preferences
  • To detect abuse and ensure bot stability
  • To debug errors and improve the bot

We never sell or share your data with third parties.

4. Data Storage & Retention

  • Logs & tickets remain until deleted by server administrators
  • Server settings remain until changed or reset
  • Temporary processing data is deleted immediately
  • Debug logs are retained short-term only

  • Cloud hosting provider — technical access only

6. Data Protection

  • Encrypted cloud servers
  • Limited developer access
  • Secure database infrastructure
  • No passwords or tokens are ever stored
  • Regular internal security checks

7. User Rights

Access

Users may request a copy of their personal data stored by the bot.

Deletion

Users may request deletion of their own personal data at any time.

Opt-Out

Users may leave the server or request admins to disable the bot’s features.
