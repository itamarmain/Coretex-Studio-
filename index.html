Coretex Discord Bot – Privacy Policy

Last Updated: 21 November 2025

Operated by Coretex Studios

1. Introduction

Coretex ("the Bot") is developed by Coretex Studios. This Privacy Policy describes how we collect, use, and protect user information. By using the Bot, you agree to this policy.

2. Data We Collect

Automatically Collected:

  • Discord User IDs
  • Server IDs
  • Channel IDs
  • Message content (only for logs, tickets, moderation, or feature processing)
  • Interaction data (commands, button usage, events)

User-Provided Data:

  • Server settings (prefix, mod role, logging channels)
  • User and guild preferences
  • Ticket information
  • Quotes and stored messages

3. Why We Collect This Data

  • To operate bot features correctly
  • To provide moderation, ticketing, logging, and automation
  • To store preferences and configurations
  • To detect abuse and maintain bot stability
  • To debug and improve the Bot

Data is never sold or shared with third parties.

4. Data Storage & Retention

  • Logs and tickets are kept until deleted by server staff.
  • Server settings remain until modified or reset.
  • Temporary data is removed immediately after use.
  • Debug logs are retained short-term only.

5. Who Has Access

  • Coretex Developers: itamar11_, ender_will_be
  • Server admins: access to their server’s data only
  • Hosting provider: technical access only

6. Data Protection

  • Encrypted cloud servers
  • Restricted developer access
  • Secure database environment
  • No storage of passwords or tokens
  • Regular internal security checks

7. User Rights

Access:

Users may request a copy of the personal data stored about them.

Deletion:

Users may request deletion of their personal data.

Opt-Out:

Users may leave the server or ask admins to disable bot features.
