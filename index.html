Coretex Bot Commands

/leaderboard

Show the server leaderboard for levels.

/rank

Check your current level and rank in the server.

/tickets

Manage ticket channels and settings.

/events

List upcoming server events.

/help

Show the help menu.

/ping

Check the bot's latency.

/invite

Get the bot invite link.

/support

Get a link to the support server.

/meme

Get a random meme.

/joke

Tell a random joke.

/8ball

Ask the magic 8-ball a question.

/roll

Roll a dice.

/ban

Ban a user from the server.

/kick

Kick a user from the server.

/mute

Mute a user for a specified time.

/warn

Warn a user.

/weather

Get weather information for a city.

/define

Get the definition of a word.

/translate

Translate text to another language.

/remind

Set a reminder for yourself or others.
